Politique de Confidentialité

Application de Gestion des Présences

Dernière mise à jour : 19 janvier 2026

1. Introduction

Cette politique de confidentialité décrit comment l'application "App Présence" collecte, utilise et protège vos données personnelles. Nous nous engageons à respecter votre vie privée et à protéger vos informations personnelles.

2. Données Collectées

L'application collecte et traite les données suivantes :

  • Informations des participants : Prénom, nom
  • Informations des clubs : Nom du club, description
  • Données de présence : Enregistrements de présence/absence aux sessions
  • Sessions : Jour de la semaine, horaires
  • Statut médical : Indicateur "malade longue durée" (optionnel)

3. Utilisation des Données

Les données collectées sont utilisées exclusivement pour :

  • Gérer les présences aux sessions
  • Générer des statistiques de présence
  • Organiser les clubs et leurs participants
  • Faciliter le partage d'informations de présence (avec votre consentement)

4. Stockage des Données

Vos données sont stockées de deux manières :

  • Localement sur votre appareil : Les données sont stockées en local via AsyncStorage pour un accès hors ligne et des performances optimales.
  • Dans le cloud (Supabase) : Les données sont synchronisées avec une base de données sécurisée hébergée par Supabase pour permettre la sauvegarde et la synchronisation entre appareils.

5. Sécurité des Données

Nous mettons en œuvre des mesures de sécurité appropriées pour protéger vos données :

  • Connexion sécurisée (HTTPS) avec Supabase
  • Authentification pour accéder aux données
  • Stockage chiffré via les mécanismes natifs des appareils
  • Pas de partage de données avec des tiers

6. Partage des Données

Vos données ne sont jamais partagées avec des tiers à des fins commerciales ou publicitaires. Le seul partage de données possible est :

  • Partage manuel : Vous pouvez choisir de partager les listes de présence ou les statistiques via les fonctionnalités de partage natives de votre appareil (WhatsApp, email, etc.). Ce partage est entièrement contrôlé par vous.

7. Vos Droits

Conformément au RGPD, vous disposez des droits suivants :

  • Droit d'accès : Vous pouvez accéder à toutes vos données via l'application
  • Droit de rectification : Vous pouvez modifier vos données à tout moment
  • Droit de suppression : Vous pouvez supprimer un participant, une session ou un club
  • Droit à la portabilité : Vos données sont stockées localement et accessibles
  • Droit d'opposition : Vous pouvez cesser d'utiliser l'application à tout moment

8. Conservation des Données

Les données sont conservées tant que vous utilisez l'application. Vous pouvez :

  • Supprimer des participants, sessions ou clubs individuellement
  • Réinitialiser les statistiques via la fonction "Démarrer l'année"
  • Désinstaller l'application pour supprimer toutes les données locales

9. Données des Mineurs

Cette application peut être utilisée pour gérer les présences de mineurs (enfants, adolescents). Il est de la responsabilité du responsable du club d'obtenir le consentement parental approprié avant d'enregistrer des données concernant des mineurs.

10. Services Tiers

L'application utilise les services suivants :

  • Supabase : Hébergement de la base de données et synchronisation. Consultez la politique de confidentialité de Supabase.
  • Expo/React Native : Framework de développement mobile. Consultez la politique de confidentialité d'Expo.

11. Modifications de la Politique

Cette politique de confidentialité peut être mise à jour occasionnellement. Toute modification sera publiée sur cette page avec une nouvelle date de mise à jour.
